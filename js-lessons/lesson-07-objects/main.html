Урок 7: Объекты

Цель урока: Научиться работать с объектами — создавать, изменять и использовать сложные структуры данных.

Что делать: Откройте файл script.js и выполните все задания.

Проверка: Откройте консоль браузера (F12), чтобы увидеть результаты выполнения.

Теория

  • Объект — коллекция пар "ключ-значение"
  • Свойства — данные внутри объекта
  • Методы — функции внутри объекта
  • Доступ к свойствам: obj.key или obj["key"]
  • this — ссылка на текущий объект внутри методов

Задания

  1. Создание и доступ к свойствам объекта
  2. Добавление и изменение свойств
  3. Методы объекта
  4. Массивы объектов
