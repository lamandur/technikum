Урок 6: Функции

Цель урока: Научиться создавать и использовать функции для переиспользования кода.

Что делать: Откройте файл script.js и выполните все задания.

Проверка: Откройте консоль браузера (F12), чтобы увидеть результаты выполнения.

Теория

  • Функция — блок кода, который можно вызывать многократно
  • Параметры — данные, которые передаются в функцию
  • Возврат значения — результат работы функции (оператор return)
  • Область видимости — переменные внутри функции доступны только внутри неё
  • Стрелочные функции — сокращённый синтаксис для создания функций

Задания

  1. Создание простых функций
  2. Функции с параметрами
  3. Функции с возвратом значения
  4. Стрелочные функции
