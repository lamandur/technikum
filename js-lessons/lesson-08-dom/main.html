Урок 8: Работа с DOM

Цель урока: Научиться взаимодействовать с элементами веб-страницы из JavaScript.

Что делать: Откройте файл script.js и выполните все задания.

Проверка: Откройте консоль браузера (F12), чтобы увидеть результаты выполнения.

Теория

  • DOM — объектная модель документа, представление страницы в виде дерева элементов
  • Получение элементов: getElementById(), querySelector(), getElementsByClassName()
  • Изменение контента: textContent, innerHTML
  • Изменение стилей: style, classList
  • Создание элементов: createElement(), appendChild()

Задания

  1. Получение элементов разными способами
  2. Изменение текста и стилей
  3. Создание и добавление новых элементов
  4. Удаление элементов
