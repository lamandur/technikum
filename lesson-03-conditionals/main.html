Урок 3: Условные операторы

Цель урока: Научиться использовать условные операторы для выполнения кода в зависимости от условий.

Что делать: Откройте файл script.js и выполните все задания.

Проверка: Откройте консоль браузера (F12), чтобы увидеть результаты выполнения.

Теория

  • if — выполняет код, если условие истинно
  • else if — проверяет дополнительные условия
  • else — выполняет код, если ни одно условие не выполнилось
  • switch — выбор одного из нескольких вариантов
  • Тернарный оператор — короткая запись условия: условие ? значение1 : значение2

Задания

  1. Простое условие с оператором if
  2. Полная конструкция if-else if-else
  3. Оператор switch
  4. Тернарный оператор
