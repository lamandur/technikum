Урок 3: Первый проект на Apache Cordova

Цель урока: Установить Apache Cordova и создать своё первое мобильное приложение.

Важно: Этот урок требует работы на компьютере с установленным Node.js. На телефоне только просмотреть нельзя!

Что такое Apache Cordova?

Apache Cordova — это фреймворк, который позволяет создавать мобильные приложения с помощью HTML, CSS и JavaScript.

Как это работает:

  • Вы пишете обычный веб-сайт (HTML/CSS/JS)
  • Cordova "оборачивает" его в нативную оболочку
  • Получается APK-файл для Android или IPA-файл для iOS
  • Приложение имеет доступ к функциям телефона: камера, геолокация, файлы и т.д.
⚠️ Важно: Для работы Cordova нужен компьютер с Windows, Linux или Mac. На телефоне установить Cordova нельзя!

Пошаговая инструкция

Шаг 1: Установка Node.js

Node.js нужен для установки Cordova через командную строку.

  1. Перейдите на официальный сайт: https://nodejs.org/
  2. Скачайте версию LTS (Long Term Support) для вашей операционной системы
  3. Установите скачанный файл (просто следуйте инструкциям установщика)
  4. После установки откройте командную строку и проверьте:
# Проверка установки Node.js
node --version

# Проверка установки npm (менеджер пакетов)
npm --version

Если вы видите номера версий (например, v18.17.0), значит всё установлено правильно!

Шаг 2: Установка Apache Cordova

Теперь устанавливаем сам Cordova через npm (менеджер пакетов Node.js).

# Установка Cordova глобально
npm install -g cordova

Что означают флаги:

  • npm — менеджер пакетов Node.js
  • install — команда установки
  • -g — установить глобально (доступно отовсюду)
  • cordova — название пакета

Дождитесь окончания установки (может занять 1-5 минут).

Проверьте установку:

cordova --version

Если вы видите номер версии (например, 12.0.0), значит Cordova установлен успешно!

Шаг 3: Создание первого проекта

Теперь создадим наш первый проект Cordova.

# Создание нового проекта
cordova create myapp com.example.myapp MyApp

Разберём параметры команды:

  • myapp — имя папки проекта (можно изменить)
  • com.example.myapp — уникальный идентификатор приложения (package name)
  • MyApp — отображаемое название приложения

После выполнения команды появится папка myapp с готовым проектом.

Шаг 4: Изучение структуры проекта

Зайдите в папку проекта и изучите её содержимое:

cd myapp
ls -la

Основные файлы и папки:

  • config.xml — конфигурационный файл приложения (название, иконки, разрешения)
  • www/ — основная папка с вашим кодом (HTML, CSS, JS)
  • platforms/ — папка для платформ (Android, iOS) — создаётся автоматически
  • plugins/ — папка для плагинов — создаётся автоматически

Ваш код находится в папке www/! Именно здесь вы будете работать.

Шаг 5: Добавление платформы

Чтобы собрать приложение для Android, нужно добавить платформу:

# Добавление платформы Android
cordova platform add android

Эта команда скачает необходимые файлы для сборки Android-приложений.

⚠️ Важно: Для полной сборки APK вам понадобится Android Studio, но для обучения мы будем использовать браузер!

Шаг 6: Запуск в браузере (для обучения)

Пока не устанавливая Android Studio, мы можем тестировать наше приложение в браузере:

# Запуск в браузере
cordova run browser

Откроется браузер с вашим приложением! Это очень удобно для обучения.

Преимущества браузерного запуска:

  • Не нужно устанавливать Android Studio (тяжелая программа)
  • Быстрая проверка кода
  • Можно сразу видеть результат

Шаг 7: Изменение кода приложения

Теперь давайте изменим наше приложение.

Откройте файл www/index.html в любом текстовом редакторе (Блокнот, VS Code и т.д.).

Найдите секцию <body> и замените содержимое на:

<body>
    <div class="app">
        <h1>Моё первое Cordova приложение!</h1>
        <p>Это работает на Cordova!</p>
        <button onclick="alert('Привет из Cordova!')">Нажми меня</button>
    </div>
</body>

Сохраните файл и обновите страницу в браузере. Вы увидите изменения!

Структура вашего проекта Cordova

myapp/
├── config.xml          # Конфигурация приложения
├── www/                # Ваш код (главная папка!)
│   ├── index.html      # Основной HTML-файл
│   ├── css/            # Стили
│   ├── js/             # JavaScript
│   └── img/            # Изображения
├── platforms/          # Платформы (создаётся автоматически)
├── plugins/            # Плагины (создаётся автоматически)
└── node_modules/       # Зависимости (создаётся автоматически)
✅ Запомните главное: Все ваши изменения делайте только в папке www/!

Задания для практики

  1. Установите Cordova на свой компьютер:
    • Установите Node.js
    • Установите Cordova через npm
    • Проверьте установку командой cordova --version
  2. Создайте первый проект:
    • Создайте проект с именем myfirstapp
    • Добавьте платформу browser
    • Запустите приложение в браузере
  3. Измените приложение:
    • Измените заголовок на своё имя
    • Добавьте кнопку, которая показывает текущее время
    • Добавьте стили (CSS) для красивого оформления
  4. Эксперимент:
    • Попробуйте добавить платформу Android (cordova platform add android)
    • Даже если Android Studio не установлена, команда должна выполниться
    • Посмотрите, какие файлы появились в папке platforms/

Частые проблемы и решения

Проблема: Команда cordova не распознаётся

Решение: Перезапустите командную строку после установки Node.js и Cordova.

Проблема: Ошибка при установке Cordova

Решение: Запустите командную строку от имени администратора (Windows) или используйте sudo (Mac/Linux).

Проблема: Приложение не запускается в браузере

Решение: Убедитесь, что вы находитесь в папке проекта и выполнили cordova platform add browser.
