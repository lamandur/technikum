Урок 4: Работа с нативными плагинами

Цель урока: Научиться подключать и использовать нативные функции телефона (камера, геолокация) через плагины Cordova.

Требования: Готовый проект Cordova из Урока 3.

Важно: Весь код в этом уроке работает только в собранном приложении (APK или через cordova run android). Тестирование в браузере невозможно.

Что такое плагины в Cordova?

Плагин — это расширение, которое даёт вашему HTML/JS-приложению доступ к нативным функциям телефона:

  • Камера
  • Геолокация
  • Файловая система
  • Контакты, аккумулятор, датчики и др.

Без плагинов ваше приложение — просто веб-страница. С плагинами — полноценное мобильное приложение!

Пошаговая инструкция

Шаг 1: Подготовка проекта

Убедитесь, что у вас есть рабочий проект Cordova из Урока 3.

Если нет — создайте новый:

cordova create mycamera com.example.mycamera MyCamera
cd mycamera
cordova platform add android

Шаг 2: Установка плагина камеры

Официальный плагин камеры от Apache:

cordova plugin add cordova-plugin-camera

Проверить установленные плагины:

cordova plugin list

Шаг 3: Код для работы с камерой

Добавьте этот код в файл www/js/index.js:

function takePicture() {
    navigator.camera.getPicture(onSuccess, onError, {
        quality: 60,
        destinationType: Camera.DestinationType.DATA_URL,
        sourceType: Camera.PictureSourceType.CAMERA
    });
}

function onSuccess(imageData) {
    const image = document.getElementById('photo');
    image.src = "data:image/jpeg;base64," + imageData;
    image.style.display = "block";
}

function onError(message) {
    alert('Ошибка камеры: ' + message);
}

Шаг 4: HTML-разметка

Замените содержимое www/index.html на:

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Камера</title>
    <style>
        body { font-family: Arial, sans-serif; padding: 20px; }
        button { padding: 12px 24px; margin: 10px; font-size: 16px;
                 background: #4CAF50; color: white; border: none;
                 border-radius: 5px; cursor: pointer; }
        img { max-width: 100%; height: auto; border-radius: 5px;
              margin-top: 15px; display: none; }
    </style>
</head>
<body>
    <h1>📸 Камера</h1>
    <button onclick="takePicture()">Сделать фото</button>
    <img id="photo">
    <script src="js/index.js"></script>
</body>
</html>

Шаг 5: Установка плагина геолокации

cordova plugin add cordova-plugin-geolocation

Добавьте в www/js/index.js:

function getLocation() {
    navigator.geolocation.getCurrentPosition(
        function(position) {
            const coords = `Широта: ${position.coords.latitude}\nДолгота: ${position.coords.longitude}`;
            alert(coords);
        },
        function(error) {
            alert('Ошибка геолокации: ' + error.message);
        }
    );
}

И добавьте кнопку в HTML:

<button onclick="getLocation()">Моё местоположение</button>

Шаг 6: Сборка и тестирование

Плагины работают только в собранном приложении:

Вариант 1 (рекомендуется):

# Подключите телефон с включённой отладкой по USB
cordova run android

Вариант 2:

cordova build android
# Установите APK из: platforms/android/app/build/outputs/apk/debug/app-debug.apk

Структура проекта

mycamera/
├── config.xml
├── www/
│   ├── index.html
│   └── js/
│       └── index.js      ← Ваш код здесь
├── plugins/              ← Плагины автоматически
└── platforms/            ← Собранные платформы
✅ Главное правило: Все изменения делайте только в папке www/. После установки плагинов пересобирать проект не нужно — Cordova сам всё подключит.

Задания

  1. Установите плагин камеры и реализуйте базовое приложение для съёмки фото.
  2. Добавьте плагин геолокации и покажите координаты в alert.
  3. Объедините обе функции: при съёмке фото также сохраняйте и показывайте координаты.
  4. Протестируйте всё на реальном устройстве через cordova run android.

Решение типичных проблем

"plugin_not_installed" → Вы тестируете в браузере. Используйте только cordova run android.

Приложение вылетает → Проверьте разрешения в настройках приложения на телефоне (Камера, Местоположение).

Нет реакции на кнопку → Убедитесь, что файл js/index.js подключен в HTML и функции определены.
