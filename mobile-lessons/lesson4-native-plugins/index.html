Урок 4: Работа с нативными плагинами

Цель урока: Научиться подключать и использовать нативные функции телефона (камера, геолокация) через плагины Cordova.

Требования: У вас должен быть готовый проект Cordova из Урока 3.

Что такое плагины в Cordova?

Плагин — это расширение, которое даёт вашему HTML/JS-приложению доступ к нативным функциям телефона:

  • Камера
  • Геолокация
  • Файловая система
  • Контакты
  • Аккумулятор
  • И многое другое!

Без плагинов ваше приложение — просто веб-страница. С плагинами — полноценное мобильное приложение!

⚠️ Важно: Плагины работают только в собранном приложении (APK или через cordova run android). В браузере они не работают!

Пошаговая инструкция

Шаг 1: Подготовка проекта

Убедитесь, что у вас есть рабочий проект Cordova из Урока 3.

Если нет — создайте новый:

cordova create mycamera com.example.mycamera MyCamera
cd mycamera
cordova platform add android

Все дальнейшие команды выполняйте внутри папки проекта!

Шаг 2: Установка плагина камеры

Официальный плагин камеры от Apache называется cordova-plugin-camera.

# Установка плагина камеры
cordova plugin add cordova-plugin-camera

Эта команда:

  • Скачивает плагин из репозитория
  • Добавляет его в ваш проект
  • Настраивает разрешения в Android/iOS

Проверить установленные плагины можно командой:

cordova plugin list

Шаг 3: Использование камеры в коде

Теперь вы можете использовать камеру в файле www/js/index.js или прямо в www/index.html.

Основная функция: navigator.camera.getPicture()

// Функция для запуска камеры
function takePicture() {
    navigator.camera.getPicture(onSuccess, onError, {
        quality: 50,
        destinationType: Camera.DestinationType.DATA_URL
    });
}

// Обработка успешного результата
function onSuccess(imageData) {
    var image = document.getElementById('myImage');
    image.src = "data:image/jpeg;base64," + imageData;
    image.style.display = "block";
}

// Обработка ошибки
function onError(message) {
    alert('Ошибка: ' + message);
}

Параметры функции:

  • quality — качество изображения (1-100)
  • destinationType — формат результата:
    • DATA_URL — base64 строка (для отображения на странице)
    • FILE_URI — путь к файлу (для сохранения)

Шаг 4: Добавление кнопки в HTML

Откройте файл www/index.html и добавьте следующий код в тело страницы:

<div class="demo-area">
    <h2>📸 Камера</h2>
    <button onclick="takePicture()">Сделать фото</button>
    <img id="myImage" class="preview" />
</div>

Не забудьте добавить функции takePicture(), onSuccess(), onError() в тег <script> или в отдельный JS-файл.

Шаг 5: Тестирование на устройстве

Плагины не работают в браузере! Нужно собрать приложение.

Вариант 1: Запуск через USB (рекомендуется)

# Подключите телефон к компьютеру
# Включите "Режим разработчика" и "Отладку по USB" на телефоне
cordova run android

Вариант 2: Сборка APK-файла

# Собрать APK-файл
cordova build android

# APK будет здесь:
# platforms/android/app/build/outputs/apk/debug/app-debug.apk

Установите APK-файл на телефон и протестируйте!

Шаг 6: Плагин геолокации

Точно так же устанавливается плагин геолокации:

# Установка плагина геолокации
cordova plugin add cordova-plugin-geolocation

Использование в коде:

function getLocation() {
    navigator.geolocation.getCurrentPosition(
        function(position) {
            alert('Широта: ' + position.coords.latitude +
                  '\nДолгота: ' + position.coords.longitude);
        },
        function(error) {
            alert('Ошибка геолокации: ' + error.message);
        }
    );
}

Структура проекта с плагинами

mycamera/
├── config.xml
├── www/
│   ├── index.html
│   └── js/
│       └── index.js
├── plugins/                 # ← Здесь хранятся все плагины
│   ├── cordova-plugin-camera/
│   └── cordova-plugin-geolocation/
└── platforms/               # ← Платформы с подключенными плагинами
✅ Запомните: После установки плагина не нужно ничего копировать вручную — Cordova всё настроит автоматически!

Задания для практики

  1. Установите плагин камеры:
    • В своём проекте из Урока 3 выполните cordova plugin add cordova-plugin-camera
    • Проверьте установку командой cordova plugin list
  2. Создайте приложение "Фотоаппарат":
    • Добавьте кнопку "Сделать фото"
    • При нажатии открывается камера
    • Сделанное фото отображается на странице
    • Протестируйте на реальном устройстве
  3. Добавьте геолокацию:
    • Установите плагин cordova-plugin-geolocation
    • Добавьте кнопку "Показать моё местоположение"
    • При нажатии показывайте координаты в alert
  4. Комбинированное приложение:
    • Создайте приложение, которое делает фото И определяет местоположение
    • Отображайте фото и координаты на одной странице
    • Подпишите фото: "Снято в [широта], [долгота]"

Частые проблемы и решения

Проблема: Камера не открывается, ошибка "plugin_not_installed"

Решение: Убедитесь, что вы тестируете на устройстве, а не в браузере.

Проблема: Приложение вылетает при запуске камеры

Решение: Проверьте, включены ли разрешения на камеру в настройках телефона.

Проблема: Геолокация не работает

Решение: Убедитесь, что GPS включён на устройстве и разрешения предоставлены.

Проблема: Долгая сборка APK

Решение: Используйте cordova run android с подключённым устройством — быстрее и удобнее.
