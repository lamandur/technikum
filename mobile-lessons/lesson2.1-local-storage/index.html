Урок 3: Работа с локальным хранилищем

Цель урока: Научиться сохранять данные в браузере с помощью localStorage и создавать приложение, которое работает без интернета.

Что делать: Изучите теорию, выполните задания и протестируйте сохранение данных.

Теория

Что такое localStorage?

localStorage — это встроенное хранилище в браузере, которое позволяет сохранять данные даже после закрытия браузера.

Особенности localStorage:

  • Данные сохраняются навсегда (пока пользователь не очистит их)
  • Хранит только строки (для объектов используйте JSON)
  • Ограничение: ~5-10 МБ на сайт
  • Работает без интернета
  • Доступен только для текущего домена

Основные методы localStorage

// Сохранить данные
localStorage.setItem('ключ', 'значение');

// Получить данные
localStorage.getItem('ключ');

// Удалить данные
localStorage.removeItem('ключ');

// Очистить всё хранилище
localStorage.clear();

// Проверить существование ключа
if (localStorage.getItem('ключ') !== null) {
    // данные существуют
}

Работа с объектами и массивами

Поскольку localStorage хранит только строки, для объектов используйте JSON:

// Сохранение объекта
const user = { name: "Иван", age: 25 };
localStorage.setItem('user', JSON.stringify(user));

// Получение объекта
const savedUser = JSON.parse(localStorage.getItem('user'));

// Сохранение массива
const notes = ["Заметка 1", "Заметка 2"];
localStorage.setItem('notes', JSON.stringify(notes));

// Получение массива
const savedNotes = JSON.parse(localStorage.getItem('notes'));

Обработка ошибок

Всегда проверяйте, поддерживается ли localStorage:

function supportsLocalStorage() {
    try {
        return 'localStorage' in window && window.localStorage !== null;
    } catch (e) {
        return false;
    }
}

if (supportsLocalStorage()) {
    // можно использовать localStorage
}

Задания

Демонстрация: Блокнот с сохранением

Сохранённые заметки:

  1. Создайте простое хранилище настроек:
    • Создайте форму с полями: Имя, Возраст, Любимый цвет
    • При изменении любого поля сохраняйте его в localStorage
    • При загрузке страницы восстанавливайте значения из localStorage
    • Добавьте кнопку "Сбросить настройки"
  2. Улучшите блокнот:
    • Добавьте дату и время создания заметки
    • Сортируйте заметки по дате (новые сверху)
    • Добавьте возможность редактирования заметки
    • Показывайте количество сохранённых заметок
  3. Создайте счётчик посещений:
    • При каждом открытии страницы увеличивайте счётчик
    • Сохраняйте счётчик в localStorage
    • Показывайте сообщение: "Вы посетили эту страницу X раз"
    • Добавьте кнопку "Сбросить счётчик"
