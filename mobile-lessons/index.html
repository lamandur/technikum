Урок 4: Доступ к функциям устройства

Цель урока: Научиться использовать встроенные функции мобильных устройств: камеру, геолокацию, уведомления.

Что делать: Изучите теорию, выполните задания и протестируйте функции на мобильном устройстве.

Теория

Доступ к функциям устройства через браузер

Современные браузеры предоставляют доступ к функциям устройства через Web APIs:

  • Камера и микрофон — MediaDevices API
  • Геолокация — Geolocation API
  • Уведомления — Notifications API
  • Акцелерометр — DeviceOrientation API

Geolocation API (геолокация)

Получение текущего местоположения пользователя:

// Получить текущее местоположение
navigator.geolocation.getCurrentPosition(
    function(position) {
        // Успех
        const lat = position.coords.latitude;
        const lng = position.coords.longitude;
        console.log(`Широта: ${lat}, Долгота: ${lng}`);
    },
    function(error) {
        // Ошибка
        console.error('Ошибка геолокации:', error.message);
    }
);

// Отслеживать изменения местоположения
const watchId = navigator.geolocation.watchPosition(successCallback, errorCallback);

// Остановить отслеживание
navigator.geolocation.clearWatch(watchId);

MediaDevices API (камера)

Доступ к камере и микрофону:

// Получить доступ к камере
navigator.mediaDevices.getUserMedia({ video: true, audio: false })
    .then(function(stream) {
        // Успех - получили видеопоток
        const video = document.getElementById('video');
        video.srcObject = stream;
    })
    .catch(function(error) {
        // Ошибка
        console.error('Ошибка камеры:', error.message);
    });

Notifications API (уведомления)

Показ системных уведомлений:

// Проверить разрешение
if (Notification.permission === 'granted') {
    // Показать уведомление
    new Notification('Заголовок', {
        body: 'Текст уведомления',
        icon: 'icon.png'
    });
} else if (Notification.permission !== 'denied') {
    // Запросить разрешение
    Notification.requestPermission().then(function(permission) {
        if (permission === 'granted') {
            new Notification('Уведомление разрешено!');
        }
    });
}

Важные моменты

  • Все эти API работают только по HTTPS (или localhost для разработки)
  • Требуется разрешение пользователя
  • На мобильных устройствах некоторые функции могут быть ограничены
  • Всегда обрабатывайте ошибки и отказы пользователя
