Урок 1: Введение в PWA (Progressive Web Apps)

Цель урока: Познакомиться с концепцией PWA и создать своё первое мобильное приложение.

Что делать: Изучите теорию ниже, выполните задания и протестируйте приложение на мобильном устройстве.

Теория

Что такое PWA?

PWA (Progressive Web App) — это веб-приложение, которое можно установить на мобильное устройство как обычное приложение.

Преимущества PWA:

  • Один код работает на Android, iOS и в браузере
  • Не требует установки Android Studio или Xcode
  • Использует знакомые технологии: HTML, CSS, JavaScript
  • Можно тестировать сразу в браузере
  • Быстрая установка без магазинов приложений

Как работает PWA?

PWA — это обычный веб-сайт с дополнительными файлами:

  • Файл манифеста (manifest.json) — описывает приложение для системы
  • Service Worker — позволяет работать офлайн (не обязателен для начала)
  • Правильный viewport — обеспечивает корректное отображение на мобильных устройствах

Как установить PWA на телефон?

  1. Откройте веб-страницу в мобильном браузере (Chrome, Safari)
  2. Нажмите на меню (три точки или "Поделиться")
  3. Выберите "Добавить на главный экран" или "Установить приложение"
  4. Приложение появится на рабочем столе!

Файл манифеста (manifest.json)

Это JSON-файл, который описывает ваше приложение:

{
    "name": "Моё приложение",
    "short_name": "Приложение",
    "start_url": "/",
    "display": "standalone",
    "background_color": "#ffffff",
    "theme_color": "#000000",
    "icons": [{
        "src": "icon-192.png",
        "sizes": "192x192",
        "type": "image/png"
    }]
}

Обязательные поля:

  • name — полное название приложения
  • short_name — короткое название (для иконки)
  • start_url — стартовая страница
  • display — режим отображения (standalone = как приложение)

Viewport Meta Tag

Этот тег обязателен для мобильных приложений:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Задания

Демонстрация

Попробуйте кнопки ниже — это уже работает как мобильное приложение!

Нажмите кнопку выше
  1. Создайте своё первое PWA:
    • Скопируйте этот HTML-файл и сохраните как my-app.html
    • Создайте файл manifest.json рядом с ним (используйте пример выше)
    • Замените название приложения на своё
    • Откройте my-app.html в браузере
    • Проверьте, что кнопки работают
    • На мобильном устройстве: установите как приложение через меню браузера
  2. Добавьте новую функцию:
    • Добавьте кнопку, которая показывает случайную цитату из массива
    • Или генерирует случайное число от 1 до 100
    • Или показывает текущую дату
  3. Настройте внешний вид:
    • Измените название приложения в manifest.json
    • Измените цвет фона (background_color)
    • Измените цвет темы (theme_color)
    • Проверьте, что цвета применяются после установки приложения
