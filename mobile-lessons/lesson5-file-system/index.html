Урок 5: Работа с файловой системой

Цель урока: Научиться сохранять данные (текст, изображения) прямо на устройство пользователя с помощью файловой системы.

Требования: Готовый проект Cordova с установленными плагинами камеры и геолокации.

Зачем нужна файловая система?

localStorage — это хорошо, но он:

  • Хранит только текст (до ~5 МБ)
  • Не может хранить изображения
  • Очищается при удалении данных браузера

Файловая система телефона позволяет:

  • Сохранять изображения, аудио, видео
  • Хранить большие объёмы данных
  • Создавать настоящие файлы (как в проводнике)
  • Делать резервные копии
⚠️ Важно: Файловая система работает только в собранном приложении (APK). В браузере — не работает!

Пошаговая инструкция

Шаг 1: Установка плагина файловой системы

Официальный плагин называется cordova-plugin-file.

# Установка плагина файловой системы
cordova plugin add cordova-plugin-file

Этот плагин даёт доступ к папкам устройства:

  • cordova.file.dataDirectory — приватная папка вашего приложения
  • cordova.file.externalDataDirectory — общая папка на SD-карте
  • cordova.file.documentsDirectory — папка "Документы"

Шаг 2: Запись текстового файла

Пример функции для сохранения текста в файл:

function saveTextToFile(text, filename) {
    window.resolveLocalFileSystemURL(
        cordova.file.dataDirectory,
        function(directory) {
            directory.getFile(filename, { create: true }, function(fileEntry) {
                fileEntry.createWriter(function(fileWriter) {
                    fileWriter.write(new Blob([text], { type: 'text/plain' }));
                    alert('Файл сохранён: ' + filename);
                });
            });
        },
        function(error) {
            alert('Ошибка записи: ' + error.code);
        }
    );
}

Как это работает:

  1. Находим папку приложения (dataDirectory)
  2. Создаём или открываем файл с указанным именем
  3. Записываем текст как Blob (двоичные данные)

Шаг 3: Чтение текстового файла

Чтение файла обратно в приложение:

function readTextFromFile(filename, callback) {
    window.resolveLocalFileSystemURL(
        cordova.file.dataDirectory,
        function(directory) {
            directory.getFile(filename, {}, function(fileEntry) {
                fileEntry.file(function(file) {
                    var reader = new FileReader();
                    reader.onloadend = function() {
                        callback(this.result);
                    };
                    reader.readAsText(file);
                });
            });
        },
        function(error) {
            alert('Файл не найден: ' + filename);
        }
    );
}

Шаг 4: Сохранение изображения

Когда вы делаете фото через камеру, вы получаете base64-строку. Её можно сохранить как файл:

function saveImageToFile(base64Data, filename) {
    window.resolveLocalFileSystemURL(
        cordova.file.dataDirectory,
        function(directory) {
            directory.getFile(filename, { create: true }, function(fileEntry) {
                fileEntry.createWriter(function(fileWriter) {
                    // Убираем префикс data:image/jpeg;base64,
                    var data = base64Data.replace(/^data:image\/jpeg;base64,/, "");
                    var binary = atob(data);
                    var array = [];
                    for (var i = 0; i < binary.length; i++) {
                        array.push(binary.charCodeAt(i));
                    }
                    var blob = new Blob([new Uint8Array(array)], { type: 'image/jpeg' });
                    fileWriter.write(blob);
                    alert('Изображение сохранено: ' + filename);
                });
            });
        },
        function(error) {
            alert('Ошибка сохранения изображения');
        }
    );
}

Шаг 5: Интеграция с камерой

Объедините камеру и файловую систему:

function takeAndSavePhoto() {
    navigator.camera.getPicture(
        function(imageData) {
            // Сохраняем изображение
            saveImageToFile(imageData, 'photo_' + Date.now() + '.jpg');

            // Также сохраняем описание
            saveTextToFile('Фото сделано: ' + new Date(),
                          'photo_' + Date.now() + '.txt');
        },
        function(message) {
            alert('Ошибка камеры: ' + message);
        },
        {
            quality: 50,
            destinationType: Camera.DestinationType.DATA_URL
        }
    );
}

Структура папок на устройстве

Ваше устройство/
└── Android/
    └── data/
        └── com.example.myapp/   ← Папка вашего приложения
            └── files/           ← Сюда сохраняются файлы
                ├── note1.txt
                ├── photo1.jpg
                └── photo2.jpg
✅ Важно: Файлы сохраняются в приватную папку приложения. Пользователь не видит их напрямую, но они остаются даже после перезапуска!

Задания для практики

  1. Установите плагин файловой системы:
    • Выполните cordova plugin add cordova-plugin-file
    • Проверьте командой cordova plugin list
  2. Создайте "Текстовый блокнот":
    • Добавьте поле ввода текста
    • Кнопку "Сохранить заметку"
    • При нажатии сохраняйте текст в файл с именем note_дата.txt
  3. Добавьте просмотр заметок:
    • Кнопка "Показать все заметки"
    • Читайте все файлы из папки и показывайте их содержимое
    • Используйте простой список
  4. Фото-блокнот (итоговое задание):
    • Сделайте фото через камеру
    • Сохраните фото как файл
    • Сохраните текстовое описание к фото
    • Покажите галерею сохранённых фото

Частые проблемы и решения

Проблема: Ошибка "FileError.NOT_FOUND_ERR"

Решение: Убедитесь, что папка существует. Используйте { create: true } при создании файла.

Проблема: Изображение не сохраняется

Решение: Проверьте, что вы правильно обрабатываете base64-строку (убираете префикс).

Проблема: Нет доступа к файлам

Решение: Убедитесь, что используете cordova.file.dataDirectory, а не другие пути.
